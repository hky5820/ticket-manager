# 🎫 티켓 보관함

내가 예매한 공연 티켓을 정리해서 안 까먹기 위한 모바일 친화 웹앱.

## 🌐 라이브 주소
**https://hky5820.github.io/ticket-manager/**
- 로그인 없이 **열면 바로 사용**. 폰·PC 어디서 열어도 **같은 목록으로 자동 동기화**.
- 폰에서 열고 **홈 화면에 추가**하면 아이콘 한 번으로 앱처럼 실행(PWA, 전체화면).

## 기능
- 예매처 선택 (티켓링크·멜론·세종·YES24·클립서비스·NOL·샤롯데 기본 제공, **+ 버튼으로 직접 추가**)
- 공연명 / 공연 날짜 / 매수 / 가격 / 자리(등급·층·구역·번호) / 메모
- 자리 여러 개 추가·삭제, 티켓 자유롭게 추가·수정·삭제
- 좌석 단위 취소·양도(부분/전체) 관리, 양도 수령액·대상·플랫폼 기록
- 상단 통계(공연 수·매수·**실지출**·**양도차익**) — 실지출/양도차익 탭 시 티켓별 산출 내역
- 캘린더/타일/리스트 3뷰 + 좌우 스와이프 전환, 캘린더 월 스와이프, 공연명/예매처 검색
- 우측 상단 **동기화 상태** 표시 · JSON **백업 내보내기/불러오기**

## 폰에서 홈 화면 앱으로 쓰기
- **아이폰(사파리):** 공유 버튼 → "홈 화면에 추가"
- **안드로이드(크롬):** ⋮ → "홈 화면에 추가"
- 이후 홈 화면 🎫 아이콘 탭 → 전체화면 앱 실행

## 구조
- 내부 설계(회계 모델·스와이프·팝업·깜빡임 방지 등)는 **[ARCHITECTURE.md](ARCHITECTURE.md)** 참고.
- **호스팅:** GitHub Pages (`hky5820/ticket-manager`, branch `main`). 자동배포.
- **백엔드:** Supabase (프로젝트 `ticket-manager`, 서울 리전). `index.html`에 URL + publishable 키 내장.
- **저장:** 표 `public.tickets`. 로그인 없이(anon) 읽기/쓰기 허용 → 열면 바로 동기화.
- 백엔드 연결 실패 시 이 기기 localStorage로 임시 저장(오프라인).
