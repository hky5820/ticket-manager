# UI 리디자인 시안 — "티켓 지갑" (KEEP)

> 2026-07-14 확정 킵. 목업: [ui-ticket-wallet.html](ui-ticket-wallet.html) (브라우저로 열면 시안 그대로 보임)
> 기존 UI(화이트+블루, Airbnb 언어)를 백지에서 재구성한 안. 기능은 현재 index.html과 1:1 동일.

## 컨셉

**"이 앱의 본질은 표(table)가 아니라 지갑 속 실물 티켓."**
카드마다 절취선·노치(구멍)·일련번호·바코드를 가진 **패스(pass)** 형태로 그린다.
Apple Wallet의 탑승권 느낌 + 공연 무대(잉크 어둠 + 스포트라이트 앰버).

## 구조 (기존 → 새)

| 기존 | 새 |
|---|---|
| 상단 세그먼트(캘린더/타일/리스트) | **하단 탭바 5슬롯**: 홈 · 캘린더 · [＋추가] · 전체 · 설정 |
| 상단 FAB(추가) + 설정 FAB | 탭바 중앙 앰버 원형 ＋버튼(스피드다이얼: 수기/사진 유지) |
| 12px 통계줄 | 잉크 헤더 안 스탯: 보유(크게) · 실지출 · 양도차익(앰버) |
| 타일/리스트 2뷰 | 홈 하나로 통합(히어로 패스 + 스텁 행). "전체" 탭 = 검색/전체 목록 |

### 홈 화면 (위→아래)
1. **잉크 헤더**: 워드마크(앰버 사각 마크 + "티켓 보관함") / 우측 검색·동기화 점(초록) / 스탯 3개(보유 20px, 실지출, 차익).
   - AI 인식 대기열 배너는 헤더 바로 아래 유지(앰버 톤).
2. **히어로 패스** (다음 공연 1장, 헤더에 -18px 겹침):
   - 상단: 예매처 색점+이름 / 우측 D-day 앰버 칩("내일")
   - 공연명 18px/800 → mono 날짜줄 `2026.07.14 TUE 20:00`
   - **절취선**(2px dashed) + 양끝 노치(paper색 원)
   - 하단: 좌석·매수 | 가격 + mono 일련번호 `NO. 2026-0714-01`
   - **바코드**(repeating-linear-gradient 세로줄)
3. **다가오는 공연 N**: 스텁 행 — 왼쪽 56px 스텁(D-day 굵게 + 날짜 mono, dashed 세로 절취선), 본문(공연명/좌석), 우측(가격/매수). 임박 스텁은 앰버 배경.
4. **지난 공연 N**: 같은 행, `opacity .62` + 스탬프. 양도는 빨간 아웃라인 "양도" 도장(rotate -6°) + 초록 `+₩차익`. 접기(더 보기 ▾).

### 티켓 상세
- 배경 = 잉크(무대 어둠), 패스 한 장 크게.
- 좌석 = 칩 나열, 양도된 좌석 칩엔 빨간 "양도" 미니 도장.
- 절취선 아래: 결제 요약(N매 결제/양도 수령/실지출·차익) + 메모(점선 박스) + 바코드.
- 하단: [수정] 아웃라인 / [공유하기] 앰버.

### 캘린더
- 흰 카드 월 그리드: 오늘=앰버 링, 선택일=잉크 채움, 지난 공연일=흐림, 예매처 색점 유지.
- 날짜 탭 → 아래에 그날 티켓 스텁 행. 월 헤더에 "공연 N건".
- 좌우 스와이프 월 이동 유지(기존 initCalSwipe 재사용).

## 앱 아이콘 (3안 — 모두 티켓 글리프: 회전 라운드 사각 + 절취선 + 노치)
- **A**: 앰버 바탕 + 잉크 티켓 ← 무난한 1순위
- **B**: 잉크 바탕 + 종이 티켓 + 앰버 스텁
- **C**: 종이 바탕 + 잉크 티켓 + 앰버 도장 점

## 구현 메모 (index.html 전환 시)
- 데이터 모델·Supabase·회계(`ticketAcct`)·AI 대기열·공유·백업 전부 그대로. **뷰 레이어만 교체.**
- 뷰 전환 스와이프: 탭바 기준으로 홈↔캘린더↔전체 스와이프 유지 가능(SWIPE_VIEWS 매핑만 변경).
- 깜빡임 방지 시그니처 dedup(`_upSig`/`_calSig`) 패턴 유지 필수 (ARCHITECTURE.md §3).
- 히어로 패스 = `renderUpcoming()`의 첫 항목 승격. 스텁 행 = listRow 대체.
- 상태 우선순위: 양도완료 > 일부양도 > 임박(D≤7, 앰버 스텁) > 예정 > 관람완료(dim).
- 아이콘: icon-180/192/512/maskable PNG 재생성 + manifest theme_color `#191b23`.
